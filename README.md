# RosRobot

## To include this package :

> `cd ~/catkin_ws/src`

> git clone <address>
  
> catkin_make

## To commit change to the project :

> git add .

> git commit -m "Commit description"

> git push <repository_add> <branch>
  
## To update the project with the modified repository :

> git pull <repository_add> <branch>
